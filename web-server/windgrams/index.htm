New Interface Features

This new windgrams interface was built to make checking conditions faster and easier, especially on mobile.

Favorites
  • Personal favorites list — pick the sites you check every day and see all their windgrams on one scrollable page.
  • Quick region presets — get started instantly by choosing your flying region. All sites in that region are included, with the most popular ones listed first.
  • Reorder, add, and remove — arrange your favorites however you like using the Edit Favorites screen.
  • Saved in your browser — your favorites persist between visits with no account required.
  • Shareable via link — generate a URL to share your favorites list with other pilots or across your own devices.
  • Explore without losing your list — browse other region presets temporarily; your saved favorites are untouched.
Custom Locations
  • Save any lat/lon as a favorite — paste a windgram URL from the map selector or enter coordinates manually.
  • On-demand windgrams — custom locations generate a fresh windgram from the latest model data, just like clicking on the map.
Viewing
  • Today / Tomorrow — one-tap switching between forecast days.
  • Responsive layout — two windgrams side-by-side on desktop, full-width on mobile. Pinch to zoom on the detail-rich images.
  • Always fresh images — automatic cache busting ensures you see the latest forecast after each 6am/6pm model run, even without reloading the page.
  • Model and duration options — switch between HRDPS/GDPS and one-day/multi-day via the Options menu. Your preference is remembered.
  • Any date — pick a specific past or future date from the Options menu.
Compatibility
  • Map selector integration — links from the Windgram Selector map open the correct windgram here.
  • Classic view — every windgram has a link to the original interface if you prefer it.
  • All original links preserved — About page, Map Overlay Plots, HRDPS docs, and donate are all still accessible.
